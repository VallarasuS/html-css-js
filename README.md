# html-css-js

## HTML Tag Structure
<> - angle bracket
`<tag-name> </tag-name>`

### Example

- `<html></html>`
- `<head></head>`
- `<body></body>`

- `<a></a>` anchor tag - link to another page
- `<p></p>` paragraph
- `<span></span>`

### Self closing tag

- `<input>`
- `<img>`
- `<br>`
- `<hr>`

### Frequently Used
- `div`
- `input`
- `a`
- `h1`, `h2`, .... `h6`
- `img`

### Attributes

- `<a href='https://google.com'></a>`
- `<input type='button'>`
- `<input id='user-name'>`
- `<h1 class='header-text'>`

href    -> attribute <br>
value   -> 'https://google.com'
